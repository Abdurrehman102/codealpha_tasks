# Music Player

Feature-rich music player with playlist and full controls.

## Features
- Play, pause, next, previous controls
- Seekable progress bar
- Volume control with mute function
- Playlist management
- Autoplay next song
- Keyboard shortcuts
- Responsive design

## Technologies
- HTML5 Audio API
- CSS3 (Animations, Gradients)
- Vanilla JavaScript

## Keyboard Shortcuts
- Space: Play/Pause
- ← / →: Previous/Next track
- ↑ / ↓: Volume up/down
